# @realfavicongenerator/grunt-real-favicon

Generate a multiplatform favicon with [RealFaviconGenerator](http://realfavicongenerator.net/).

## Getting Started

So you want to use this plugin? Don't write any code. Instead:

- Go to [RealFaviconGenerator for Grunt](https://next.realfavicongenerator.net/favicon/grunt), submit your original image and craft your icons:

![RealFaviconGenerator's favicon editor](https://cloud.githubusercontent.com/assets/423852/11236405/b08079a2-8dd9-11e5-8d42-c49943fd1e1f.png)

- On the result page, follow the instructions to setup your favicon in your Grunt project:

![Result page](https://cloud.githubusercontent.com/assets/423852/11236407/b4408e74-8dd9-11e5-861d-a2be47b7dce2.png)
